When you do Buffer.alloc / Buffer.from, Node doesnâ€™t always ask the OS for memory directly.

The size of the page cache is not fixed at 4 GB â€” it dynamically grows/shrinks depending on available system RAM.
Example: if you have 16 GB RAM, the OS might use 12 GB for page cache if needed.
